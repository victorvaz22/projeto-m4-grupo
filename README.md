
# Projeto do módulo 4 - Banco de dados.

Montagem de um dashboard com base no conjunto de banco de dados disponibilizado pela Resilia. O banco de dados escolhido foi o Game Of Thrones.

# Utilizamos:

Power BI, MySQL e Xhamps.
